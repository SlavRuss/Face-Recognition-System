Система распознавания лиц для поиска сотрудников

Описание 

Веб-приложение для поиска сотрудников по фотографии лица. Использует нейросетевые модели для генерации эмбеддингов лиц и поиска наиболее похожих сотрудников в базе данных.

Функциональность

- Аутентификация пользователей с сессиями и cookies
- Поиск сотрудников по загруженной фотографии
- История поисков для каждого пользователя
- Админ-панель для управления пользователями и сотрудниками
- Добавление сотрудников с автоматической генерацией эмбеддингов

Технологии

- Backend: Python, Streamlit
- Распознавание лиц: DeepFace (Facenet512, RetinaFace)
- База данных: PostgreSQL
- Аутентификация: Сессии с cookies, хэширование паролей
- Обработка изображений: OpenCV, Pillow

 Установка и запуск
 
 Требования
- Python 3.8+
- PostgreSQL 12+
- Git

Запуск приложения

- pip install -r requirements.txt
- streamlit run streamlit_app.py

Приложение будет доступно по адресу: http://localhost:8501
